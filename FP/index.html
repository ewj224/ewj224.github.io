Enoch Jo: Web Dev Final Project

AMBI is music application designed to pair audio with visual input. The app features multiple rooms, each representing a certain genre/type of music with a customizable avatar that is able to perform actions in the room. As it is designed to be on an app interface, the application would hold the shape of a cube with each side reveal a different type of room. AMBI serves the purpose of further enhancing one's experience with music while also creating avenues to customize the experience to one's liking. In the future, I hope to even add a VR element to the application so that one could even travel in these spaces in Virtual Reality.
